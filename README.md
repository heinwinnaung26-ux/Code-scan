# Code Scan (WIFI SCAN BYPASS)

This tool is designed for scanning and bypassing WiFi authentication.

## Installation in Termux (proot-distro Ubuntu)

1. **Enter Ubuntu environment:**
   ```bash
   proot-distro login ubuntu
   ```

2. **Clone the repository:**
   ```bash
   git clone https://github.com/heinwinnaung26-ux/Code-scan
   cd Code-scan
   ```

3. **Run the setup script:**
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

4. **Run the scanner:**
   ```bash
   python3 Code_scan.py
   ```

## Features
- Automated Captcha solving using ddddocr
- Multi-threaded scanning (Concurrency)
- Obfuscated code for security
- Support for multiple scan modes (Numeric, Alphanumeric, etc.)
